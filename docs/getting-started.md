#  Getting Started

# Instalación

Clona el repositorio:

```bash
git clone https://github.com/Juanyepes04/Proyecto-Codigo-Limpio.git
cd Proyecto-Codigo-Limpio
```

Instala dependencias:

```bash
uv sync
```

---

# Ejecutar el proyecto

```bash
uv run main.py --help
```

---

# Ejemplo

```bash
uv run main.py create-user --name "Juan"
```
