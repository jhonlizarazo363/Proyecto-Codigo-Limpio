# Comandos

# Crear usuario

```bash
uv run main.py create-user --name "Juan" --email "juan@email.com"
```

Permite registrar un nuevo usuario en el sistema.

---

# Listar usuarios

```bash
uv run main.py list-users
```

Muestra todos los usuarios registrados.

---

# Eliminar usuario

```bash
uv run main.py delete-user --id 1
```

Elimina un usuario por su ID.
